## Specification


create a python library that return number and return "Fizz" for multiple of 3 and "Buzz" for multiple of 5
"FizzBuzz" for multiple of 3 and 5


> exemple :

```s
1 => 1
2 => 2
3 => "Fizz"
4 => 4
5 => "Buzz"
6 => "Fizz"
7 => 7
10 => "Buzz"
...
15 => "FizzBuzz"
```
